## âš™ï¸ Installation
To set up Fairleap AI API backend, start by clone the repository
   ```sh
   git clone https://github.com/Fairleap-AI/fairleap-api
   cd fairleap-api
   ```

To set up the Backend locally, run:
   ```sh
   npm run dev
   ```

To deploy using Docker compose, run:
   ```sh
   docker compose up
   ```
